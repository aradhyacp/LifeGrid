1

Copy URL

Configure your wallpaper above and copy the generated URL.

2

Install MacroDroid

Download and install MacroDroid from the Play Store and open it.

3

Set a New Trigger

Tap Trigger → Date/Time

  • Time: 00:01:00
  • Select all days (Mon–Sun)
  • Turn Use Alarm OFF
4

Configure Actions

4.1

Download Wallpaper

  • Tap Actions → +
  • Select Web Interaction → HTTP Request
  • Choose GET method
  • Paste your copied URL
  • Enable Block next action until complete
  • Enable Save HTTP response to file
  • Choose a folder & enter filename: wallpaper.png
Remember this folder for the next step
4.2

Set Wallpaper

  • Tap Actions → Device Settings → Set Wallpaper
  • Choose Image → Tap OK
  • Select Home Screen & Lock Screen
  • Choose Dynamic File Name
  • Select the same folder as before
  • Enter the same filename: wallpaper.png
5

Finalize & Test

Tap the three dots (⋮) in the top-right corner and select Test Macro to verify everything works.
